## [有效的字母异位词](https://leetcode-cn.com/problems/valid-anagram/)

给定两个字符串 _s_ 和 _t_ ，编写一个函数来判断 _t_ 是否是 _s_ 的字母异位词。

**示例 1:**

`**输入:** _s_ = "anagram", _t_ = "nagaram"
**输出:** true
`

**示例 2:**

`**输入:** _s_ = "rat", _t_ = "car"
**输出:** false`

**说明:**  

你可以假设字符串只包含小写字母。

**进阶:**  

如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？
